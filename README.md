# AI Data Intelligence Assistant

An AI-powered analytics platform that automates data quality assessment, visualization, insight generation, and machine learning recommendations.

## Features

* CSV and Excel Upload
* Data Quality Analysis
* Missing Value Detection
* Duplicate Record Detection
* Outlier Detection
* Histogram Visualization
* Box Plot Visualization
* Correlation Heatmap
* AI Insight Generation
* ML Task Recommendation
* Dataset Download
* PDF Report Generation
* Dataset Chat Assistant

## Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Seaborn
* Matplotlib
* Scikit-Learn
* FPDF

## How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
